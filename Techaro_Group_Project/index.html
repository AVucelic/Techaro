Unix is a computer Operating System[1] which is capable of handling activities from multiple users at the same time. Oldest operating system that has ever existed. Run multiple application programs simultaneously. Developed in the 1960s at AT&T Bell Labs by Ken Thompson and Dennis Ritchie command-line driven and has a hierarchical file system

[1]]operating system- suite of programs which make the computer work.

Main Features

  1. simplicity
  2. portability -able to run on many different systems
  3. multitasking
  4. multi user capabilities
  5. extensive library of software
  6. hierarchical file system
    • Most popular varieties of UNIX are Sun Solaris, GNU/Linux, and MacOS X
