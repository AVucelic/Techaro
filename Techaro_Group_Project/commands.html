Commands

$ ls -> ls(list) - list files and directories

% ls -a -> list all files and directories

% mkdir -> nameofdirectoryïƒ  mkdir (make directory)

% cd nameofdirectory -> cd(change directory)- change to named directory

% cd. -> The current directory (.) -> (.) means the current directory

% cd .. -> The parent directory (..)

% pwd pwd (print working directory)

~ (your home directory) -> can be used to specify paths starting at your home directory

Manipulating files:

% cp -> cp (copy)- cp file1 file2 - copy file1 and call it file2

% mv -> mv (move) - move or rename file1 to file2

%rm file remove a file

% mkdir nameofdirectory -> mkdir (make directory)

%Rmdir directory remove a directory

%cat file display a file

%less file display a file a page at a time

%head file display the first few lines of a file

%tail file display the last few lines of a file

%grep 'keyword' file search a file for keywords

Some of the other options of grep are:

  • -v display those lines that do NOT match
  • -n precede each matching line with the line number
  • -c print only the total count of matched lines

%wc file count number of lines/words/characters in file

Command > file redirect standard output to a file

command >> file append standard output to a file

command < file redirect standard input from a file

command1 | command2 pipe the output of command1 to the input of command2

cat file1 file2 > file0 concatenate file1 and file2 to file0

sort sort data

%who list users currently logged in

WILDCARDS:

* match any number of characters

? match one character (e.g ?ouse can be house,mouse)

%man command read the online manual page for a command

%whatis command brief description of a command

%apropos keyword match commands with keyword in their man pages
